# Dataset Guide

## 목표 데이터 수
- 최소: 15장
- 권장: 25~40장

## 좋은 학습 이미지 조건
- 아라찌가 크게 보일 것
- 캐릭터 단독 비중이 높을 것
- 그림체가 일관될 것
- 표정과 포즈가 다양할 것
- 저해상도/압축 손상이 심하지 않을 것

## 꼭 포함하면 좋은 유형
- 정면
- 측면
- 웃는 표정
- 멍한 표정
- 앉은 포즈
- 소품 든 포즈
- 전신
- 상반신 클로즈업

## 빼는 게 좋은 유형
- 캐릭터가 너무 작음
- 다른 캐릭터가 더 눈에 띔
- 텍스트/워터마크가 큼
- 배경이 지나치게 복잡함
- 그림체가 너무 다름

## 폴더 사용 규칙
- `datasets/raw/`: 원본 전부
- `datasets/selected/`: 후보군
- `datasets/train/`: 실제 학습 투입본
- `datasets/eval/`: 검증용 홀드아웃
- `datasets/captions/`: 캡션 텍스트

## 추천 작업 순서
1. raw에 전부 모으기
2. selected로 1차 선별
3. train/eval 분리
4. 캡션 작성
5. 부족한 유형 체크
