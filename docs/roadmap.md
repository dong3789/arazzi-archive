# Roadmap

## 0. 목표 정의
- 목표: 아라찌 캐릭터를 로컬에서 지속 생성 가능한 상태로 복원
- 성공 기준:
  - 같은 캐릭터로 인식되는 결과물 다수 생성 가능
  - 표정/상황/소품 변경 가능
  - Mac mini에서 반복 가능한 워크플로우 확보

## 1. 빠른 검증
- 대표 이미지 3~5장 선정
- ComfyUI + IPAdapter 테스트
- 캐릭터 일관성 평가

## 2. 데이터셋 구축
- raw 이미지 수집
- selected 이미지 선별
- train/eval 분리
- captions 작성

## 3. LoRA 1차 학습
- 작은 설정으로 시작
- 재현성 확인
- 실패 결과 기록

## 4. 보정
- 부족한 각도/표정 데이터 보강
- 캡션 수정
- 재학습

## 5. 운영화
- 자주 쓰는 프롬프트 정리
- LoRA + IPAdapter 기본 워크플로우 고정
- 결과물 관리 규칙 정리
